# sundial-org/awesome-openclaw-skills (427 stars)

## 问题与解决方案

OpenClaw 生态中的 913 个技能分散在各处，开发者难以发现高质量技能。该项目通过每日自动抓取 ClawHub 下载量数据，按 20 个类别（Agent Core、开发工具、社交媒体、金融交易等）整理最受欢迎的技能，提供一站式技能发现平台。

## 关键特性

- **913 个精选技能** — 覆盖 Agent 核心（ByteRover 知识管理、self-improving-agent）、生产力工具、开发者工具、Web 搜索、社交媒体、内容创作等 20 个类别
- **下载量排序** — 每个技能显示实际下载数（如 ByteRover 14,376 次），帮助用户快速识别社区验证的高质量技能
- **一键安装** — 统一使用 `npx sundial-hub add <name>` 命令安装，无需手动查找仓库地址
- **每日更新** — 自动同步 ClawHub 最新数据，确保技能列表始终反映生态最新趋势
- **结构化元数据** — 每个技能包含安装名、描述、下载量、ClawHub 链接，便于程序化访问
- **分类导航** — 按功能域分类（Agent Core & Memory、Productivity & Tasks、Developer Tools 等），支持快速定位需求
