# openclaw-zero-token (1,455 stars)

## 问题与解决方案
使用 AI 模型需要购买 API Token，存在成本和信用卡绑定门槛。该项目通过浏览器自动化捕获会话凭证，实现免费访问 ChatGPT、Claude、Gemini、DeepSeek、Qwen、Doubao、Kimi、GLM、Grok、Manus 等主流 AI 平台，消除 API Token 成本。

## 关键特性
- 支持 14+ AI 平台的免费访问（DeepSeek、Qwen 国际/中国、Kimi、Claude Web、Doubao、ChatGPT Web、Gemini Web、Grok、GLM、Manus）
- 通过 Playwright 浏览器自动化捕获会话凭证
- 所有模型支持本地工具调用（exec、read_file、list_dir、browser、apply_patch）
- AskOnce 功能：一次查询多个 AI 模型并对比响应
- 凭证本地存储，无泄漏风险
- 提供一键安装脚本和完整文档
