# openclaw-zero-token (1,455 stars)

## Problem & Solution
Using AI models requires purchasing API Tokens, creating cost and credit card binding barriers. This project uses browser automation to capture session credentials, enabling free access to mainstream AI platforms including ChatGPT, Claude, Gemini, DeepSeek, Qwen, Doubao, Kimi, GLM, Grok, and Manus, eliminating API Token costs.

## Key Features
- Free access to 14+ AI platforms (DeepSeek, Qwen International/China, Kimi, Claude Web, Doubao, ChatGPT Web, Gemini Web, Grok, GLM, Manus)
- Session credential capture via Playwright browser automation
- Local tool calling support for all models (exec, read_file, list_dir, browser, apply_patch)
- AskOnce feature: query multiple AI models at once and compare responses
- Credentials stored locally with no leakage risk
- One-click installation script and complete documentation
