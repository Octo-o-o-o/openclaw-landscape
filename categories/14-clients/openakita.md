# OpenAkita (1,114 stars)

## 问题与解决方案

OpenAkita 解决了传统 AI 助手"只会聊天不会做事"的痛点。通过多 Agent 协作架构，将复杂任务分解给专业化 Agent 并行执行（编码 Agent 写代码、写作 Agent 写文档、测试 Agent 验证），配合 ReAct 推理引擎（Think → Act → Observe 三阶段循环）和 89+ 工具集成，实现了从对话到实际执行的闭环。

## 关键特性

- **多 Agent 协作系统** — 专业化 Agent 并行委派、自动交接、故障转移，实时可视化仪表板
- **Plan Mode 任务分解** — 复杂任务自动拆解为分步计划，实时进度追踪，失败自动回滚
- **ReAct 推理引擎** — 显式三阶段循环（思考-行动-观察），支持检查点/回滚、循环检测、策略切换
- **零门槛 GUI 配置** — 5 分钟从安装到对话，全程图形化向导，无需命令行（下载 → 安装 → 向导 → API Key → 开聊）
- **89+ 工具 + 技能市场** — 16 类工具（Shell/文件/浏览器/桌面/搜索/定时任务/MCP），一键安装技能或 AI 即时生成
- **30+ LLM 提供商 + 6 IM 平台** — DeepSeek/Qwen/Claude/GPT/Gemini 智能故障转移，Telegram/飞书/企微/钉钉/QQ 原生集成
- **三层记忆系统** — 工作记忆 + 核心记忆 + 动态检索，7 种记忆类型，AI 驱动提取与复盘
- **主动引擎 + 自我进化** — 问候/任务跟进/闲聊/晚安，根据反馈调整频率；每日自检修复、失败根因分析、自动技能生成
