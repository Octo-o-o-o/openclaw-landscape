# OpenAkita (1,114 stars)

## Problem & Solution

OpenAkita addresses the pain point of traditional AI assistants that "can only chat but can't act." Through a multi-Agent collaboration architecture, it breaks down complex tasks to specialized Agents executing in parallel (coding Agent writes code, writing Agent writes docs, testing Agent verifies), combined with a ReAct reasoning engine (Think -> Act -> Observe three-phase loop) and 89+ tool integrations, achieving a closed loop from conversation to actual execution.

## Key Features

- **Multi-Agent Collaboration System** — Specialized Agent parallel delegation, automatic handoff, failover, with real-time visualization dashboard
- **Plan Mode Task Decomposition** — Complex tasks automatically broken into step-by-step plans, real-time progress tracking, automatic rollback on failure
- **ReAct Reasoning Engine** — Explicit three-phase loop (Think-Act-Observe) with checkpoint/rollback support, loop detection, and strategy switching
- **Zero-Barrier GUI Configuration** — 5 minutes from installation to conversation, fully graphical wizard, no command line needed (Download -> Install -> Wizard -> API Key -> Chat)
- **89+ Tools + Skill Marketplace** — 16 tool categories (Shell/File/Browser/Desktop/Search/Cron/MCP), one-click skill install or AI-generated on demand
- **30+ LLM Providers + 6 IM Platforms** — DeepSeek/Qwen/Claude/GPT/Gemini with intelligent failover; Telegram/Feishu/WeCom/DingTalk/QQ native integration
- **Three-Layer Memory System** — Working memory + core memory + dynamic retrieval, 7 memory types, AI-driven extraction and review
- **Proactive Engine + Self-Evolution** — Greetings/task follow-up/casual chat/goodnight messages with feedback-adjusted frequency; daily self-check repair, failure root cause analysis, automatic skill generation
