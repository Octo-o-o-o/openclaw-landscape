## Overview

A one-click installation script suite for running OpenClaw on Android phones. No proot required, no full Linux installation needed — runs natively on Termux. Supports turning old phones into 24/7 AI assistants.

## Core Features

- **One-Click Install** — Single command completes the entire installation in Termux
- **No Linux Required** — Does not depend on proot or full Linux distributions
- **Layered Installation** — L1 core infrastructure -> L2 platform runtime -> L3 optional tools
- **8-Step Installation Flow** — Environment check -> Platform selection -> Optional tools -> Core installation -> Platform configuration -> Verification
- **Process Keep-Alive** — Supports keeping OpenClaw running in the background
- **PC Remote Access** — Access the phone's Dashboard from a PC browser
- **Multi-Device Management** — Supports managing OpenClaw across multiple Android devices
- **Lightweight Updater** — 5-step incremental update process
- **Local LLM** — Supports running local LLMs on Android

## Installation Components

- **Core Infrastructure**: Node.js, Git, Python, etc.
- **Platform Runtime**: OpenClaw Gateway
- **Optional Tools**: Install as needed
