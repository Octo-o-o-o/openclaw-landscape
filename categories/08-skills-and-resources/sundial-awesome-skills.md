# sundial-org/awesome-openclaw-skills (N/A stars)

## Problem & Solution

The 913 skills in the OpenClaw ecosystem are scattered everywhere, making it difficult for developers to discover high-quality skills. This project automatically scrapes ClawHub download data daily and organizes the most popular skills into 20 categories (Agent Core, Developer Tools, Social Media, Finance & Trading, etc.), providing a one-stop skill discovery platform.

## Key Features

- **913 curated skills** — Covering Agent core (ByteRover knowledge management, self-improving-agent), productivity tools, developer tools, web search, social media, content creation, and 20 other categories
- **Download count sorting** — Each skill displays actual download numbers (e.g., ByteRover 14,376 downloads), helping users quickly identify community-verified high-quality skills
- **One-click installation** — Unified `npx sundial-hub add <name>` command for installation, no need to manually find repository addresses
- **Daily updates** — Automatically syncs the latest ClawHub data, ensuring the skill list always reflects the latest ecosystem trends
- **Structured metadata** — Each skill includes install name, description, download count, and ClawHub link for programmatic access
- **Category navigation** — Organized by functional domain (Agent Core & Memory, Productivity & Tasks, Developer Tools, etc.) for quick needs-based navigation
