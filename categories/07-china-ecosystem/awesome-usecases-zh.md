# Awesome OpenClaw Usecases ZH (1,556 stars)

## Problem & Solution
Addresses the core bottleneck in OpenClaw adoption: not the technical barrier, but users not knowing "what they can do with an AI Agent." Through 40 verified real-world use cases (with Chinese ecosystem adaptations), it teaches users step-by-step how to automate work and life with AI, bridging the cognitive gap from "knowing about OpenClaw" to "actually using it."

## Core Architecture
- **Scenario classification system** — 6 major categories (social media, creative building, DevOps, productivity, research & learning, financial trading) + a dedicated China-specific use cases section
- **Unified use case template** — 5 standard sections (pain points, feature list, required skills, configuration steps, practical tips) to reduce learning costs
- **Chinese ecosystem adaptation** — 13 China-specific use cases covering domestic tools such as Feishu (Lark)/DingTalk/WeCom/Xiaohongshu (RED)/AKShare/Baidu Index
- **Difficulty levels** — One star: copy-paste ready; Two stars: simple configuration; Three stars: requires technical background

## Key Features
- **40 real-world use cases** — From Reddit summaries and YouTube content pipelines to self-healing servers and multi-agent collaboration systems
- **Beginner-friendly** — Core concept comparison table (workspace/soul/playbook/memory/skills/tools/channels) + three-step getting started guide
- **Copy-ready prompts** — Each use case provides directly copyable English prompts with Chinese explanations for comprehension
- **Chinese adaptation solutions** — Morning briefing push to Feishu (Lark)/DingTalk, A-share financial report tracking (AKShare), multi-channel customer service (WeCom/Douyin/Xiaohongshu)
- **Security reminders** — Clearly notes that third-party skills are not audited, prompting users to review source code and permissions
