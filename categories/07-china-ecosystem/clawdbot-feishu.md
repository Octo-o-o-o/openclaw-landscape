# Clawdbot-Feishu (4,189 stars)

## Problem & Solution

Clawdbot-Feishu solves the integration challenge of connecting OpenClaw to enterprise collaboration scenarios. Through a Feishu (Lark) plugin, it seamlessly integrates OpenClaw Agent capabilities into the enterprise IM platform, supporting private chat, group chat @mentions, message subscriptions, file uploads, and other enterprise-level interaction scenarios, lowering the barrier for enterprise users to adopt AI Agents.

## Key Features

- **Full Feishu (Lark) integration**: Supports private chat, group chat @mentions, message sending, media upload/download, message editing/recall, and emoji reactions
- **Tool permission system**: Fine-grained permission control (read-only/read-write), supporting Feishu Docs, Cloud Drive, Knowledge Base, Bitable (multi-dimensional tables), Tasks, Group Chat, and other APIs
- **Flexible configuration**: Supports `requireMention` to control group chat trigger mode, optional user info parsing (display name attribution)
- **Enterprise-grade permissions**: Follows the principle of least privilege; read-only permissions suffice for basic functions, with read-write permissions enabled on demand
- **Chinese community support**: Wiki provides configuration tutorials, FAQs, and usage tips; Discussions provide community support
- **Structured feedback**: Issue Form templates + Discussions routing to improve issue resolution efficiency
