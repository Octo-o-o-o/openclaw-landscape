# awesome-openclaw-skills-zh (2,911 stars)

## Problem & Solution
The official OpenClaw skill library is in English, and skills are scattered and hard to discover. This project translates the official skill library and organizes it by use case (office automation, system tools, DevOps, etc.), supports Chinese natural language invocation, and reduces the cost of skill discovery and usage for Chinese users.

## Key Features
- Translated from the official Clawdbot skill library with continuous synchronization
- Organized by use case: office automation, system tools, DevOps, AI & search, social media, etc.
- Supports Chinese natural language invocation
- Covers 80,000+ community skills including email management, calendar scheduling, document processing, CRM, code development, CI/CD, database management, and more
- Provides core feature descriptions and official links for each skill
