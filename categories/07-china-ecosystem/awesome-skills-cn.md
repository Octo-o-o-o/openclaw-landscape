# awesome-openclaw-skills-CN (51 stars)

## Problem & Solution

The official OpenClaw skill registry (ClawHub) contains 5,705 skills but is plagued with spam, duplicate entries, and descriptions unfriendly to Chinese developers. This project manually curates 2,868 high-quality skills organized into 18 categories, and provides guides for integrating Chinese LLMs along with recommended skill lists for Chinese developers.

## Key Features

- **Manual curation for quality assurance** — Filters out 2,748 low-quality entries (spam accounts, cryptocurrency, duplicates, malicious content)
- **Chinese LLM integration guides** — Provides detailed integration documentation for Chinese models including DeepSeek, Tongyi Qianwen, GLM, Moonshot, Doubao, and more
- **Chinese ecosystem recommendations** — Curates dedicated skills for Chinese platforms such as Feishu (Lark), WeCom, Xiaohongshu (RED), Zhihu, and Bilibili
- **18-category taxonomy** — Organized by function (coding, DevOps, browser automation, AI/LLM, marketing, productivity, etc.)
- **Security notices** — Clearly states the list is manually curated rather than security-audited, and recommends using Snyk and Agent Trust Hub for security scanning
- **Community contribution mechanism** — Provides contribution guidelines, encouraging Chinese developers to submit new skills and improvement suggestions
