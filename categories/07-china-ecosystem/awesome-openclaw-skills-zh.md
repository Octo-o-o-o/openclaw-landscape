# awesome-openclaw-skills-zh (3128 stars)

## Problem & Solution
An official Chinese repository for OpenClaw skills, translated from Clawdbot's official skills, making it easier for Chinese users to integrate and call skills using natural language.

## Key Features
- **Categorized Skills** — Organized by use cases (development, productivity, etc.).
- **Natural Language** — Adapted for Chinese prompt engineering.
- **Community Contributions** — Open for the Chinese community to submit localized skills.

## Tech Stack
- Markdown, Prompt Engineering
