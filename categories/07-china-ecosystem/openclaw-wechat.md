# openclaw-wechat (1,347 stars)

## Problem & Solution
OpenClaw lacks WeChat connectivity, unable to cover the primary communication platform for Chinese users. This project implements a stable connection between OpenClaw and personal WeChat through a WeChat Channel Plugin, supporting private chat, group chat, text, images, and other message types, solving the AI assistant integration problem for Chinese users.

## Key Features
- Supports private chat and group chat (@bot)
- Supports text and image messages
- QR code login flow
- Multi-account support (work account, personal account)
- Webhook configuration for cloud server deployment
- Supports iPad and Mac device types
- Stable connection through proxy service (proxyUrl)
