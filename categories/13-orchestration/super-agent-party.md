# super-agent-party (1,834 stars)

## Problem & Solution
AI Agents lack desktop-grade interaction experiences and multi-role collaboration capabilities. super-agent-party provides an all-in-one AI desktop companion supporting VRM desktop pets, a task center (background computer control), multi-role group chat (Tavern character cards + long-term memory), instant messaging bots (QQ/Feishu/DingTalk/Telegram/Discord), live streaming bots (Bilibili/YouTube/Twitch), and an AI browser.

## Key Features
- VRM desktop pet (supports custom VRM models)
- Task center (Agent automatically controls computer in the background to complete advanced tasks)
- Multi-role group chat (supports Tavern character cards and long-term memory)
- Instant messaging bots (one-click deployment to QQ/Feishu/DingTalk/Telegram/Discord/Slack)
- Live streaming bots (one-click deployment to Bilibili/YouTube/Twitch)
- AI browser (Agent with built-in browser, supports automated control)
- Extension system (supports installing and creating extensions, such as Galgame extension)
- Developer-friendly (provides OpenAI API interface and MCP interface)
