# trohitg/MachinaOS (18 stars)

## Problem & Solution

N8N provides powerful workflow automation but lacks AI Agent capabilities; OpenClaw provides an Agent framework but lacks visual orchestration and enterprise-grade integration. MachinaOS merges both, offering a "visual workflow builder + AI Agent team collaboration" hybrid solution with enhanced operational visibility and control granularity.

## Key Features

- **Visual Workflow Builder** — Drag-and-drop interface connecting AI models, services, and triggers; build complex automations without coding
- **13 Specialized Agents** — Pre-built Agents for Android control, coding, web operations, task management, social media, travel, payments, and other verticals
- **Agent Team Collaboration** — Supports "Team Lead Agent" coordinating multiple specialized Agents, automatically delegating based on task context (delegation tools)
- **Enterprise-Grade Integration** — Native support for Google Workspace (Gmail, Calendar, Drive, Sheets), WhatsApp, Twitter/X, Telegram, Android device control
- **Multi-Model Support** — Integrates OpenAI, Claude, Gemini, Groq, and 200+ OpenRouter models with model switching and cost optimization
- **Real-Time Monitoring** — Team Monitor visualizes the real-time operational status and task flow of Agent teams
