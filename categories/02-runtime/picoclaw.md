## Introduction

PicoClaw is an ultra-lightweight personal AI assistant inspired by Nanobot, rebuilt from scratch in Go. The entire architecture migration and code optimization were driven by AI Agent self-bootstrapping, achieving extreme resource efficiency.

**Core Selling Point**: Runs on $10 hardware, memory usage <10MB, starts in just 1 second.

## Core Features

- **Ultra Lightweight**: <10MB memory usage, 99% smaller than OpenClaw
- **Extremely Low Cost**: Runs on $10 hardware, 98% cheaper than a Mac mini
- **Lightning Fast Startup**: 400x faster startup, 1-second start on a 0.6GHz single core
- **True Portability**: Single binary file, supports RISC-V, ARM, MIPS, x86
- **AI Self-bootstrapped Development**: 95% of core code generated by Agent, with human review and optimization

## Supported Platforms

- Chat platforms: Telegram, Discord, WhatsApp, etc.
- LLM: Multi-provider support
- Security sandbox: Supports isolated execution
- Heartbeat tasks (scheduled tasks)
- Docker Compose deployment
