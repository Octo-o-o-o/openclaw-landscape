# secure-openclaw (1,358 stars)

## Problem & Solution
Users need 24/7 available AI assistants but lack cross-platform messaging access, tool access control, and application integration capabilities. Secure OpenClaw provides secure tool access via Composio Tool Router, persistent memory, scheduled reminders, and 500+ app integrations, running on WhatsApp, Telegram, Signal, iMessage, and other platforms.

## Key Features
- Supports WhatsApp, Telegram, Signal, iMessage, and other messaging platforms
- Composio Tool Router provides 500+ app integrations (Gmail, Slack, GitHub, etc.)
- Tool approval mechanism: sensitive operations require user confirmation
- Persistent memory system: cross-session memory of user preferences and context
- Scheduled reminders and task scheduling
- Supports both Claude Code and Opencode as AI providers
- Interactive CLI menu: terminal chat, gateway startup, adapter configuration, etc.
