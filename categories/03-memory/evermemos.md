# EverMemOS (2,520 stars)

## Problem & Solution

EverMemOS addresses the long-term memory problem for AI Agents. Traditional Agents (including OpenClaw) have memory limited to single sessions or short-term context, unable to persist and evolve across sessions, platforms, or LLMs. EverMemOS provides an OS-level memory layer supporting 24/7 running Agents with continuous learning, memory sharing, and cross-platform migration, with integrations for OpenClaw, Live2D characters, TEN Framework, and other scenarios.

## Key Features

- **Cross-Platform Memory**: Unified memory layer across LLMs (Claude/GPT/Gemini) and platforms (OpenClaw/Dify/LangChain)
- **Continuous Learning**: 24/7 running Agents can continuously accumulate memory, maintaining context across restarts and sessions
- **Memory Evolution**: Supports incremental memory updates, associative retrieval, and forgetting mechanisms that simulate human memory characteristics
- **Multiple Storage Backends**: Supports MongoDB (document storage), Elasticsearch (full-text search), and Milvus (vector search)
- **OpenClaw Integration**: Provides an OpenClaw Agent Memory Plugin for seamless integration into the OpenClaw ecosystem
- **Memory Genesis Competition 2026**: Hosts a memory application competition with three tracks (Agent + Memory / Platform Plugins / OS Infrastructure)
