# openclaw-manager (1219 stars)

## Problem & Solution
A GUI manager for OpenClaw that helps users manage their running instances, skills, and configuration from a friendly dashboard instead of using the CLI.

## Key Features
- **Instance Management** — Start, stop, restart, and monitor OpenClaw nodes.
- **Skill Configuration** — UI for adding and configuring OpenClaw skills.
- **Log Viewer** — Real-time logging output from the OpenClaw service.

## Tech Stack
- React, Tailwind, Node.js
