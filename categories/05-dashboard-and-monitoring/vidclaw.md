# VidClaw (139 stars)

## Problem & Solution
OpenClaw Agents running on remote servers lack observability -- users cannot understand Agent execution status and behavior in real time. VidClaw transforms each Agent run into an observable, auditable, and shareable video recording through live screen recording and playback capabilities.

## Key Features
- **Kanban Task Management** — Backlog -> Todo -> In Progress -> Done drag-and-drop task flow, with Agents automatically picking up tasks via heartbeat or cron
- **Real-Time Usage Tracking** — Parses token usage and cost estimates from session records, with progress bars matching Anthropic rate limit windows
- **Model Hot-Switching** — Switch Claude models directly from the dashboard, hot-reloaded via OpenClaw config listener
- **Activity Calendar View** — Monthly Agent activity view parsed from memory files and task history
- **Workspace File Browser** — File browser with markdown preview, syntax highlighting, and download support
- **Skill Manager** — View all bundled/workspace skills, enable/disable, and create custom skills
