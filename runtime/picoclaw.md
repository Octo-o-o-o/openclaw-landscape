## 简介

PicoClaw 是一款受 Nanobot 启发、用 Go 语言从头重构的超轻量级个人 AI 助手。通过 AI Agent 自举（self-bootstrapping）方式驱动整个架构迁移和代码优化，实现了极致的资源效率。

**核心卖点**：能在 $10 的硬件上运行，内存占用 <10MB，启动仅需 1 秒。

## 核心特性

- 🪶 **超轻量级**：<10MB 内存占用，比 OpenClaw 小 99%
- 💰 **极低成本**：$10 硬件即可运行，比 Mac mini 便宜 98%
- ⚡️ **闪电启动**：启动速度快 400 倍，0.6GHz 单核 1 秒启动
- 🌍 **真正的可移植性**：单一二进制文件，支持 RISC-V、ARM、MIPS、x86
- 🤖 **AI 自举开发**：95% 核心代码由 Agent 生成，人工审查优化

## 支持平台

- 聊天平台：Telegram、Discord、WhatsApp 等
- LLM：多 Provider 支持
- 安全沙盒：支持隔离执行
- 心跳任务（定时任务）
- Docker Compose 部署

## 技术亮点

- 单一 Go 二进制文件，跨架构部署
- 支持在旧 Android 手机上运行
- 支持 Raspberry Pi 和 RISC-V 开发板
- 已有 143+ 贡献者参与开发
