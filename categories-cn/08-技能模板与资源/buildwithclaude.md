# Build with Claude (2,565 stars)

## 问题与解决方案
Claude Code 生态缺乏统一的插件发现和分发平台，开发者难以找到合适的扩展工具。Build with Claude 提供了一个集中式的插件市场和发现平台，索引了 20k+ 社区插件、4,500+ MCP 服务器和 1,100+ 插件市场，通过 Web UI 和 CLI 命令实现一键安装。

## 关键特性
- 117 个专业 Agent（Python、Go、DevOps、Security 等领域专家）
- 175 个 Slash 命令用于自动化（/commit、/docs、/tdd）
- 28 个事件驱动 Hook（通知、git、格式化）
- 26 个可复用 Skill 和 50 个分类插件包
- Web UI 支持搜索、过滤、一键复制安装命令
- 通过 `/plugin marketplace add` 集成到 Claude Code
