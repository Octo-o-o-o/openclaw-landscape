# market-sentry (31 stars)

## 问题与解决方案

投资者需要同时监控 A 股、美股、加密货币等多资产类别，手动跟踪异动和生成日报耗时且容易遗漏。该 Skill 提供多市场统一监控，支持实时异动检测（价格阈值、Z-score 统计异常、成交量激增）、每日简报生成和飞书推送，并通过 EvidencePack 机制确保数据可审计。

## 关键特性

- **多市场支持** — A 股（K 线、主力资金、量比、公告、板块）、美股（行情、新闻、SEC 文件）、加密货币（价格、成交量、链上数据）
- **三层异动检测** — 价格阈值（如 5 分钟涨跌 >2%）、Z-score 统计异常、成交量激增
- **叙事式简报** — 生成自然语言日报（类似研报），而非简单的数据罗列
- **飞书双模式推送** — Mode A（App 机器人直接消息）、Mode B（Webhook 富交互卡片，推荐）
- **EvidencePack 审计** — 每份简报生成 JSON 证据包，包含所有原始 API 响应，防止幻觉
- **静默监控模式** — 异动时才推送告警，日常保持静默，降低噪音
