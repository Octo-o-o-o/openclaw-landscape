# secure-openclaw (1,358 stars)

## 问题与解决方案
用户需要 24x7 可用的 AI 助手，但缺乏跨平台消息接入、工具访问控制和应用集成能力。Secure OpenClaw 通过 Composio Tool Router 提供安全的工具访问、持久化内存、定时提醒和 500+ 应用集成，在 WhatsApp、Telegram、Signal、iMessage 等平台上运行。

## 关键特性
- 支持 WhatsApp、Telegram、Signal、iMessage 等多消息平台
- Composio Tool Router 提供 500+ 应用集成（Gmail、Slack、GitHub 等）
- 工具审批机制：敏感操作需用户确认
- 持久化内存系统：跨会话记忆用户偏好和上下文
- 定时提醒和任务调度
- 支持 Claude Code 和 Opencode 两种 AI 提供商
- 交互式 CLI 菜单：终端聊天、网关启动、适配器配置等
