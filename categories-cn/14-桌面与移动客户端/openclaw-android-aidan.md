## 简介

在 Android 手机上一键运行 OpenClaw 的安装脚本套件。无需 proot、无需完整 Linux 安装，原生 Termux 运行。支持将旧手机变成 24/7 AI 助手。

## 核心特性

- **一键安装** — 单条命令在 Termux 中完成全部安装
- **无需 Linux** — 不依赖 proot 或完整 Linux 发行版
- **分层安装** — L1 核心基础设施 → L2 平台运行时 → L3 可选工具
- **8 步安装流程** — 环境检查 → 平台选择 → 可选工具 → 核心安装 → 平台配置 → 验证
- **进程保活** — 支持后台保持 OpenClaw 运行
- **PC 远程访问** — 从 PC 浏览器访问手机上的 Dashboard
- **多设备管理** — 支持管理多台 Android 设备上的 OpenClaw
- **轻量更新器** — 5 步增量更新流程
- **本地 LLM** — 支持在 Android 上运行本地 LLM

## 安装组件

- **核心基础设施**：Node.js、Git、Python 等
- **平台运行时**：OpenClaw Gateway
- **可选工具**：根据需求选装
