# openclaw-lark (884 stars)

## Problem & Solution
飞书官方出品的 OpenClaw 飞书/Lark Channel 插件。解决 OpenClaw 智能体与飞书/Lark 通信平台的无缝连接问题，降低了企业的接入成本。

## Key Features
- **飞书集成** — 全面支持飞书机器人的收发消息能力
- **富文本与卡片** — 原生支持图片、文件、和复杂的交互式消息卡片

## Tech Stack
- TypeScript, Node.js, Lark Open API
