# awesome-openclaw-skills-CN (51 stars)

## 问题与解决方案

OpenClaw 官方技能注册表（ClawHub）包含 5,705 个技能，但存在大量垃圾内容、重复条目和非中文开发者友好的描述。该项目人工整理 2,868 个高质量技能，按 18 个分类组织，并提供国产大模型接入指南和中文开发者推荐技能列表。

## 关键特性

- **人工筛选质量保证** — 过滤掉 2,748 个低质量条目（垃圾账号、加密货币、重复、恶意内容）
- **国产模型接入指南** — 提供 DeepSeek、通义千问、GLM、Moonshot、豆包等国产模型的详细接入文档
- **中文生态推荐** — 整理飞书、企业微信、小红书、知乎、B 站等国内平台的专用 Skill
- **18 个分类体系** — 按功能分类（编码、DevOps、浏览器自动化、AI/LLM、营销、生产力等）
- **安全提示** — 明确说明列表为人工整理而非安全审计，推荐使用 Snyk 和 Agent Trust Hub 进行安全扫描
- **社区共建机制** — 提供贡献指南，鼓励中文开发者提交新技能和改进建议
