# trohitg/MachinaOS (18 stars)

## 问题与解决方案

N8N 提供强大的工作流自动化但缺乏 AI Agent 能力，OpenClaw 提供 Agent 框架但缺乏可视化编排和企业级集成。MachinaOS 将两者融合，提供"可视化工作流构建器 + AI Agent 团队协作"的混合方案，同时增强了操作可见性和控制粒度。

## 关键特性

- **可视化工作流构建器** — 拖拽式界面连接 AI 模型、服务和触发器，无需编码即可构建复杂自动化
- **13 个专业 Agent** — 预置 Android 控制、编码、Web 操作、任务管理、社交媒体、旅行、支付等垂直领域 Agent
- **Agent 团队协作** — 支持"Team Lead Agent"协调多个专业 Agent，自动根据任务上下文智能委派（delegation tools）
- **企业级集成** — 原生支持 Google Workspace（Gmail、Calendar、Drive、Sheets）、WhatsApp、Twitter/X、Telegram、Android 设备控制
- **多模型支持** — 集成 OpenAI、Claude、Gemini、Groq 及 200+ OpenRouter 模型，支持模型切换和成本优化
- **实时监控** — Team Monitor 可视化展示 Agent 团队的实时操作状态和任务流转
