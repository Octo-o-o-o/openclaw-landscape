# PowerMem (489 stars)

## 问题与解决方案

AI 应用需要持久化记忆历史对话、用户偏好和上下文信息，但全上下文方案成本高、响应慢。PowerMem 结合向量检索、全文搜索、图数据库的混合架构，引入 Ebbinghaus 遗忘曲线理论，在 LOCOMO 基准测试中实现 48.77% 准确率提升、91.83% 延迟降低、96.53% Token 成本削减。

## 关键特性

- **混合检索架构** — 向量检索 + 全文搜索 + 图数据库三通道召回，通过 LLM 构建知识图谱并支持多跳图遍历
- **Ebbinghaus 遗忘曲线** — 基于认知科学的记忆保留率计算和时间衰减权重，自动优先检索近期和相关记忆
- **智能记忆提取** — 通过 LLM 自动从对话中提取关键事实、检测重复、更新冲突信息、合并相关记忆
- **多 Agent 支持** — 独立记忆空间、跨 Agent 记忆共享与协作、基于 scope 的细粒度权限控制
- **多模态支持** — 自动将图像和音频转换为文本描述存储，支持文本 + 图像 + 音频混合内容检索
- **Sub Stores 分区** — 数据分区管理 + 自动查询路由，显著提升超大规模数据的查询性能
- **多种集成方式** — Python SDK、CLI (`pmem`)、MCP Server、HTTP API Server
